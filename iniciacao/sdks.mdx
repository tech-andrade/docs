
```bash Python

```

```bash Go

```

## Configuração inicial

<CodeGroup>

```javascript Node.js
import { DelFinance } from '@delfinance/sdk';

const client = new DelFinance({
  environment: 'sandbox', // ou 'production'
  clientId: process.env.DELFINANCE_CLIENT_ID,
  clientSecret: process.env.DELFINANCE_CLIENT_SECRET,
  certPath: './certificado.crt',
  keyPath: './chave-privada.key',
});
```

```python Python
from delfinance import DelFinance
import os

client = DelFinance(
    environment="sandbox",
    client_id=os.getenv("DELFINANCE_CLIENT_ID"),
    client_secret=os.getenv("DELFINANCE_CLIENT_SECRET"),
    cert_path="./certificado.crt",
    key_path="./chave-privada.key",
)
```

```go Go
import "github.com/delfinance/sdk-go"

client, err := delfinance.New(delfinance.Config{
    Environment:  "sandbox",
    ClientID:     os.Getenv("DELFINANCE_CLIENT_ID"),
    ClientSecret: os.Getenv("DELFINANCE_CLIENT_SECRET"),
    CertPath:     "./certificado.crt",
    KeyPath:      "./chave-privada.key",
})
```

</CodeGroup>

## Exemplo de uso

<CodeGroup>

```javascript Node.js
// Listar contas
const contas = await client.accounts.list();

// Criar pagamento (idempotência automática)
const pagamento = await client.payments.create({
  valor: 100.0,
  destinatario: '12345678900',
});
```

```python Python
# Listar contas
contas = client.accounts.list()

# Criar pagamento
pagamento = client.payments.create(
    valor=100.00,
    destinatario="12345678900",
)
```

```go Go
contas, err := client.Accounts.List(ctx)

pagamento, err := client.Payments.Create(ctx, &delfinance.PaymentInput{
    Valor:        100.00,
    Destinatario: "12345678900",
})
```

</CodeGroup>

## Recursos dos SDKs

- ✅ Renovação automática de tokens OAuth 2.0
- ✅ Configuração de mTLS transparente
- ✅ Geração automática de chaves de idempotência
- ✅ Retries com backoff exponencial
- ✅ Tratamento tipado de erros
- ✅ Tipagem completa (TypeScript, Python type hints, Go structs)
