Esta página funciona como uma referência rápida para os ambientes e as URLs base. O fluxo inicial de onboarding, com o passo a passo para começar, está em [Primeiros passos](/iniciacao/primeiros-passos).

## Ambientes

| Ambiente | Uso | URL base |
| --- | --- | --- |
| Sandbox | testes e validação | `https://apisandbox.delbank.com.br` |
| Produção | operações reais | `https://api.delbank.com.br` |

## URLs base

As chamadas devem ser feitas via HTTPS e seguem versionamento por path:

```text
https://apisandbox.delbank.com.br/v1/{recurso}
https://api.delbank.com.br/v1/{recurso}
```
