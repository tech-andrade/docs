## O que os SDKs fazem por você

| Recurso | Sem SDK | Com SDK |
|---|---|---|
| Autenticação OAuth2 | Manual | Automática |
| Renovação de token expirado | Manual | Automática |
| Configuração mTLS | Manual | Uma linha |
| `IdempotencyKey` | Você gera | Automático |
| Retry com backoff | Você implementa | Incluído |
| Tratamento de erros | Você parseia | Tipado |

## Requisitos gerais

Todos os SDKs precisam de:

- **API Key** e **Account ID** — obtidos no Portal do Desenvolvedor
- **Certificado mTLS** (`.crt`) e **chave privada** (`.key`) — gerados no portal e necessários para ambientes produtivos
- Acesso à URL correta: `apisandbox.delbank.com.br` (sandbox) ou `api.delbank.com.br` (produção)
