# Buscar Split de Pagamentos

## Buscar Todos os Split de Pagamentos configurados

### Request

```curl curl
GET 'https://apisandbox.delbank.com.br/baas/api/v1/split-payments/configurations'
```

#### Headers

| Name              | Description                                                     |
| :---------------- | :-------------------------------------------------------------- |
| x-delbank-api-key | **Obrigatório**. Chave de API                                           |
| x-delfinance-account-id | **Obrigatório**. O número da conta bancária Delfinance |

## Buscar configuração de Split de Pagamento por ID

### Request

```curl curl
GET 'https://apisandbox.delbank.com.br/baas/api/v1/split-payments/configurations/{id}'
```

#### Headers

| Name              | Description                                                     |
| :---------------- | :-------------------------------------------------------------- |
| x-delbank-api-key | **Obrigatório**. Chave de API                                           |
| x-delfinance-account-id | **Obrigatório**. O número da conta bancária Delfinance |

#### Path Parameteres
| Name              | Description                                                     |
| :---------------- | :-------------------------------------------------------------- |
| Id | **Obrigatório**. O ID da configuração de Split de Pagamento a ser buscado                                           |

### Response

Em caso de sucesso, você receberá:
