---
title: "Remover configuração de Split"
api: "DELETE /baas/api/v1/split-payments/configurations/{splitPaymentId}"
description: "Remova uma configuração de Split de Pagamento e todas as suas instruções."
---

<Note>
Para o passo a passo completo, veja o guia [Remover Split de Pagamentos](/guias/split-pagamentos/remover).
</Note>

Remove uma configuração de Split de Pagamento por completo, incluindo todas as instruções associadas. A operação é irreversível.

<Warning>
Após a remoção, os pagamentos seguintes recebidos na conta original deixam de ser divididos.
</Warning>

## Headers

<ParamField header="x-delbank-api-key" type="string" required>
  Sua API key.
</ParamField>

<ParamField header="x-delfinance-account-id" type="string" required>
  Número da conta Delfinance.
</ParamField>

## Path

<ParamField path="splitPaymentId" type="string" required>
  ID da configuração de Split de Pagamento a ser removida.
</ParamField>

## Response

Retorna `204 No Content` em caso de sucesso.

<RequestExample>
```bash cURL
curl --location --request DELETE 'https://apisandbox.delbank.com.br/baas/api/v1/split-payments/configurations/15' \
--header 'x-delbank-api-key: {{apiKey}}' \
--header 'x-delfinance-account-id: {{accountId}}'
```
</RequestExample>

(corpo vazio)
