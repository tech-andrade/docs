---
title: "Cancelar QR Code com Vencimento"
api: "PATCH /pix/qrcode/due-date/:transactionIdentifier/cancel"
description: "Cancela uma cobrança com vencimento, impedindo que ela seja paga."
---

<Note>
Para o passo a passo completo de uso, veja o guia [Cobrança com Vencimento](/guias/pix/receber-pix/cobranca-com-vencimento).
</Note>

Cancela uma cobrança com vencimento ativa. Após o cancelamento, qualquer tentativa de pagamento será recusada.

## Headers

<ParamField header="x-delbank-api-key" type="string" required>
  Sua API key.
</ParamField>

<ParamField header="x-delfinance-account-id" type="string" required>
  Número da conta Delfinance.
</ParamField>

## Path Parameters

<ParamField path="transactionIdentifier" type="string" required>
  Identificador do QR Code: pode ser o `correlationId` ou o `transactionId`.
</ParamField>

## Response

Um `status code 202` indica sucesso no cancelamento. **Nenhum corpo de resposta é retornado.**

<RequestExample>
```bash cURL
curl -X 'PATCH' \
  'https://apisandbox.delbank.com.br/baas/api/v2/pix/qrcode/due-date/{{transactionIdentifier}}/cancel' \
  -H 'x-delbank-api-key: {{apiKey}}' \
  -H 'x-delfinance-account-id: {{accountId}}'
```
</RequestExample>

<ResponseExample>
```text 202
(sem corpo de resposta)
```
</ResponseExample>
