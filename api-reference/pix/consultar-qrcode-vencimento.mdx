---
title: "Consultar QR Code com Vencimento"
api: "GET /pix/qrcode/due-date/:transactionIdentifier"
description: "Consulta os dados de uma cobrança com vencimento e o pagamento associado, quando realizado."
---

<Note>
Para o passo a passo completo de uso, veja o guia [Cobrança com Vencimento](/guias/pix/receber-pix/cobranca-com-vencimento).
</Note>

## Headers

<ParamField header="x-delbank-api-key" type="string" required>
  Sua API key.
</ParamField>

<ParamField header="x-delfinance-account-id" type="string" required>
  Número da conta Delfinance.
</ParamField>

## Path Parameters

<ParamField path="transactionIdentifier" type="string" required>
  Identificador do QR Code: pode ser o `correlationId` ou o `transactionId`.
</ParamField>

## Response

<ResponseField name="correlationId" type="string">
  Identificador de correlação definido na criação.
</ResponseField>

<ResponseField name="transactionId" type="string">
  Identificador único do QR Code.
</ResponseField>

<ResponseField name="amount" type="number">
  Valor efetivo da cobrança (após abatimento, se aplicado).
</ResponseField>

<ResponseField name="originalAmount" type="number">
  Valor original enviado na criação.
</ResponseField>

<ResponseField name="dueDate" type="string">
  Data de vencimento.
</ResponseField>

<ResponseField name="maxDaysOverdue" type="number">
  Dias após o vencimento em que o QR Code ainda é válido para pagamento.
</ResponseField>

<ResponseField name="status" type="string">
  Status atual do QR Code. Valores possíveis: `ACTIVE`, `CONCLUDED`, `CANCELLED`.
</ResponseField>

<ResponseField name="revision" type="number">
  Número de revisões da cobrança.
</ResponseField>

<ResponseField name="taxes" type="object">
  Configuração de encargos e descontos definida na criação.
</ResponseField>

<ResponseField name="payer" type="object">
  Dados do pagador (`name`, `document` mascarado).
</ResponseField>

<ResponseField name="address" type="object">
  Endereço da transação.
</ResponseField>

<ResponseField name="payloadPix" type="string">
  Payload Pix (BR Code) em texto — o "copia e cola".
</ResponseField>

<ResponseField name="createdAt" type="string">
  Data e hora de criação, em formato ISO 8601.
</ResponseField>

<ResponseField name="updatedAt" type="string">
  Data e hora da última atualização.
</ResponseField>

<ResponseField name="expiresAt" type="string">
  Data e hora limite para pagamento.
</ResponseField>

<ResponseField name="additionalInfos" type="array">
  Informações adicionais definidas na criação.
</ResponseField>

<RequestExample>
```bash cURL
curl --location 'https://apisandbox.delbank.com.br/baas/api/v2/pix/qrcode/due-date/{{transactionIdentifier}}' \
--header 'x-delbank-api-key: {{apiKey}}' \
--header 'x-delfinance-account-id: {{accountId}}'
```
</RequestExample>
