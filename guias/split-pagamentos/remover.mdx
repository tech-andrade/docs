Você pode remover uma configuração de Split de Pagamento inteira, encerrando a divisão automática para a conta de recebimento, ou remover apenas uma instrução específica, mantendo a configuração e as demais instruções ativas.

<Warning>
A remoção é irreversível. Após remover uma configuração ou uma instrução, os pagamentos seguintes recebidos na conta deixam de ser divididos para aquele beneficiário.
</Warning>

## Remover uma configuração por completo

Use `DELETE /baas/api/v1/split-payments/configurations/:splitPaymentId` para remover a configuração e todas as suas instruções.

<CodeGroup>
```bash cURL
curl --location --request DELETE 'https://apisandbox.delbank.com.br/baas/api/v1/split-payments/configurations/15' \
--header 'x-delbank-api-key: {{apiKey}}' \
--header 'x-delfinance-account-id: {{accountId}}'
```
</CodeGroup>

A resposta de sucesso retorna `204 No Content`. A partir dessa chamada, os pagamentos recebidos na conta original deixam de ser divididos.

## Remover uma instrução específica

Use `DELETE /baas/api/v1/split-payments/configurations/:splitPaymentId/beneficiaries/:splitPaymentBeneficiaryId` para remover apenas uma instrução, mantendo a configuração e as demais instruções ativas.

<CodeGroup>
```bash cURL
curl --location --request DELETE 'https://apisandbox.delbank.com.br/baas/api/v1/split-payments/configurations/15/beneficiaries/18' \
--header 'x-delbank-api-key: {{apiKey}}' \
--header 'x-delfinance-account-id: {{accountId}}'
```
</CodeGroup>

A resposta de sucesso também retorna `204 No Content`.
