A API Pix da DelFinance permite integrar sua aplicação ao arranjo de pagamentos instantâneos do Banco Central de forma segura e escalável.

## Conceitos fundamentais

| Conceito | Descrição |
| --- | --- |
| **Pagador** | Pessoa ou empresa que inicia o Pix (envia o valor). |
| **Recebedor** | Pessoa ou empresa que recebe o valor. |
| **Chave Pix** | Identificador do recebedor: CPF/CNPJ, e-mail, telefone, chave aleatória ou EVP. |
| **EndToEndId** | Identificador único da transação no SPI/Bacen. Formato: `E` \+ ISPB \+ timestamp \+ sufixo. |
| **TransactionId (TxId)** | Identificador da cobrança/transação na DelFinance. Usado para rastrear a operação. |
| **QR Code** | Representação visual dos dados do Pix. Pode ser estático (reutilizável) ou dinâmico (uso único). |
| **Webhook** | Notificação HTTP enviada pela DelFinance quando um evento ocorre (recebimento, falha, devolução). |
| **Devolução** | Operação que retorna valor total ou parcial de um Pix recebido ao pagador original. |
